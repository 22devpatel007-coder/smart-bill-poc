# Smart POS Installation Guide

## Step 1: Extract Files
Copy the `SmartPOS` folder to to your desired location (e.g., `C:\SmartPOS`).

## Step 2: First-Run Setup
Run the `first_run_setup.exe` (or `python first_run_setup.py`).
This will initialize the database with default GST rates and an admin account. 
You will be prompted to enter your:
- Shop Name
- Address
- GST Number
- Phone
- Printer Port

## Step 3: Start Smart POS
Double-click `SmartPOS.exe` to start the application.

### Default Login
- **Username:** admin
- **Password:** admin123

## Notes
- To configure the thermal printer, attach the USB and confirm Windows sees the printer. Leave port empty to auto-detect.
